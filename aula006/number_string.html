Tipos Primitivos

Os tipos de dados primitivos são os tipos básicos que devem ser implementados por todas as linguagens de programação:

No JavaScript, existem 6 tipos primitivos:

  • String = Sequência de caracteres.
    Number = Valor numérico do objeto.
    Boolean = Numérico com ponto flutuante( 5.5 , 2.2 , etc...)
    Null
    undefined.
    Symbol (novo no ECMAScript (en-US) 6)
