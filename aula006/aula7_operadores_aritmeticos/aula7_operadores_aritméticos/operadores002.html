Operadores Relacionais

Nos operadores relacionais, o resultado de uma operação sempre será "true" ou "false".Vejamos:


5 > 2 ( 5 é maior do que 2 ) true

7 < 4 ( 7 é menor do que 4 ) false

8 >= 8 ( 8 é maior ou igual a 8 ) true

5 == 5 ( 5 é igual a 5 ) true

4 != 4 ( 4 é diferente de 4 ) false
